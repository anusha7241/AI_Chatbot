# AI Chatbot 🤖
A simple rule-based AI chatbot built using Python. This chatbot interacts with users and responds to basic queries using keyword matching.

# Description

This project is a beginner-friendly chatbot that simulates conversation with users. It uses simple Natural Language Processing (NLP) techniques like keyword matching to generate responses.

# Features

* Interactive chat system
* Predefined responses
* Keyword-based matching
* Exit option for ending chat
